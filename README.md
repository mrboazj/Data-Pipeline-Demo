# data-pipeline-demo

A practical Python ETL project that simulates a payment transaction pipeline.

## Problem
Raw transaction data is often inconsistent and not analysis-ready.

## Solution
This project:
- ingests raw transaction data
- validates and cleans records
- transforms data into analytics-ready outputs
- generates summary reports

## How to run
```bash
pip install -r requirements.txt
python src/pipeline.py
```
